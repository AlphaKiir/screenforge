# ScreenForge

**Windowed-mode fix + auto-scanner for any PC game.**

ScreenForge auto-detects your installed games (Steam, Epic Games, GOG, custom folders) and applies a clean borderless-windowed mode with centered window, cursor clipping, and in-game overlay (Shift+F1) featuring FPS / GPU stats and live resolution control.

## Features

- **Auto-scanner** — detects games in Steam libraries, Epic Games, GOG Galaxy, Ubisoft, and custom roots (`D:\Games`, etc.)
- **Universal engine support** — Unity, Unreal Engine 4 & 5, Source, Godot, CryEngine, RE Engine, 4A Engine
- **Borderless windowed mode** — clean fullscreen look without GPU crash risk
- **Cursor clipping** — keeps the mouse inside the game window on multi-monitor setups
- **Live resolution control** — change game resolution in real time from the app or overlay
- **Auto-center on launch** — fixes games that open in the wrong corner
- **In-game overlay (Shift+F1)** — quick toggle buttons, resolution dropdown, FPS / GPU stats
- **Dedicated fixes** — Metro 2033 Redux, Metro Last Light Redux, Metro Exodus, Metro Exodus EE, Resident Evil 2, 3 and 4 Remake
- **8 languages** — EN, FR, ES, DE, IT, PT, RU, PL
